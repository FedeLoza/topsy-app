# Topsy 📀

A music album explorer based on app like Topsters 3 built with vanilla JavaScript and the Deezer API. Deployed in Vercel.

## About

I created this project to practice:

- API integration
- asynchronous JavaScript
- dynamic DOM manipulation
- CSS Grid layouts
- carousel interactions
- handling CORS issues with serverless functions

The application allows users to search for albums and display them dynamically in a responsive interface.

## Features

- Search albums by artist or album name
- Dynamic album carousel
- Responsive CSS Grid layout
- Horizontal drag-scroll interaction
- Deezer API integration
- Serverless API proxy using Vercel Functions

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Deezer API
- Vercel deploy

## Challenges

One of the biggest challenges was dealing with CORS restrictions from the Deezer API.

To solve this, I implemented a serverless proxy using Vercel Functions so the frontend could communicate safely with the API.
